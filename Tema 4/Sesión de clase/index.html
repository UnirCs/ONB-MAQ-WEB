1. Introducción

Hasta ahora hemos aprendido a seleccionar elementos usando selectores simples como clases, IDs y etiquetas. Pero ¿qué pasa cuando queremos seleccionar un elemento específico que está dentro de otro? ¿O cuando queremos ser muy precisos sobre qué elementos modificar?

En esta sección aprenderemos dos cosas importantes:

📚 Lo que aprenderemos:
  • Selectores Combinados: Cómo seleccionar elementos basándonos en su relación con otros elementos
  • Posicionamiento: Cómo decidir exactamente dónde debe aparecer un elemento en la página

Imagina que tienes una casa (tu página HTML) con varias habitaciones (divs), muebles (elementos) y decoraciones (estilos). Los selectores combinados te permiten decir "quiero cambiar todos los cuadros que están en la sala", y el posicionamiento te permite decir "quiero que este cuadro esté exactamente aquí, en esta esquina".

2. Selectores Combinados

Los selectores combinados nos permiten apuntar a elementos que tienen una relación específica con otros elementos. Hay cuatro tipos principales:

2.1 Selector Descendente (Espacio)

Selecciona todos los elementos que están dentro (a cualquier nivel de profundidad) de otro elemento.

📍 Sintaxis
padre descendiente { }

Código CSS y HTML:

article p {
  color: blue;
}
<article>
  <p>Párrafo 1</p>
  <div>
    <p>Párrafo 2</p>
  </div>
</article>

Resultado Visual:

Párrafo 1 ✓

Párrafo 2 (dentro de div) ✓

Ambos párrafos son azules porque son descendientes del article

2.2 Selector Hijo (>)

Selecciona solo los elementos que son hijos directos de otro elemento (no nietos, bisnietos, etc.).

📍 Sintaxis
padre > hijo { }

Código CSS y HTML:

ul > li {
  color: green;
}
<ul>
  <li>Elemento 1</li>
  <li>Elemento 2</li>
</ul>

Resultado Visual:

  • Elemento 1 ✓ (hijo directo)
  • Elemento 2 ✓ (hijo directo)
  • Elemento 3 ✗ (nieto, NO se aplica)
📍 Descendente: article p

Selecciona TODOS los párrafos dentro del article.

Incluye párrafos en cualquier nivel de profundidad.

Más general y amplio.

👆 Hijo: article > p

Selecciona SOLO párrafos directamente dentro del article.

Los párrafos que estén dentro de otros elementos no se seleccionan.

Más específico y restrictivo.

2.3 Selector Adyacente (+)

Selecciona un elemento que está **inmediatamente después** de otro elemento al mismo nivel (hermano).

📍 Sintaxis
elemento1 + elemento2 { }

Código CSS y HTML:

h2 + p {
  font-weight: bold;
  color: red;
}
<h2>Mi Título</h2>
<p>Primer párrafo</p>
<p>Segundo párrafo</p>

Resultado Visual:

Mi Título

Primer párrafo ✓ (inmediatamente después)

Segundo párrafo ✗ (no es adyacente al h2)

2.4 Selector Hermano General (~)

Selecciona todos los elementos que vienen después de otro elemento y están al mismo nivel (hermanos).

📍 Sintaxis
elemento1 ~ elemento2 { }

Código CSS y HTML:

h2 ~ p {
  background: yellow;
}
<h2>Mi Título</h2>
<p>Primer párrafo</p>
<p>Segundo párrafo</p>
<p>Tercer párrafo</p>

Resultado Visual:

Mi Título

Primer párrafo ✓

Segundo párrafo ✓

Tercer párrafo ✓

Todos los párrafos hermanos después del h2 tienen fondo amarillo

3. Posicionamiento CSS

El posicionamiento controla exactamente dónde se coloca un elemento en la página. La propiedad CSS es position.

3.1 Position: Static (Estático) - El valor por defecto

Los elementos con position: static se colocan en el flujo normal del documento, uno después de otro. Este es el comportamiento por defecto.

📍 Características de Static
  • Es el posicionamiento por defecto
  • Los elementos se colocan en el orden que aparecen en el HTML
  • Las propiedades top, bottom, left, right no funcionan
  • Las propiedades z-index no funcionan

3.2 Position: Relative (Relativo)

Con position: relative, el elemento se coloca relativamente a su posición normal. El espacio original del elemento se mantiene.

📍 Características de Relative
  • Se desplaza de su posición normal usando top, bottom, left, right
  • El espacio original que ocupaba el elemento sigue "reservado" (otros elementos no lo ocupan)
  • z-index funciona con relative
  • Es útil para pequeños ajustes de posición

Código CSS:

.caja {
  position: relative;
  top: 20px;
  left: 30px;
  background: #764ba2;
  width: 200px;
  height: 100px;
}

Resultado Visual:

Caja desplazada

El borde punteado muestra la posición original. La caja se mueve pero su espacio se reserva.

3.3 Position: Absolute (Absoluto)

Con position: absolute, el elemento se coloca relativamente a su contenedor padre más cercano que tenga un posicionamiento relative, absolute, o fixed. Se saca del flujo normal.

📍 Características de Absolute
  • Se saca completamente del flujo normal del documento
  • Se posiciona relativamente a su contenedor padre posicionado más cercano
  • Si no hay padre posicionado, se posiciona relativamente a la ventana del navegador
  • El espacio que ocupaba NO se reserva (otros elementos pueden ocuparlo)
  • z-index funciona con absolute
  • Muy útil para superponer elementos

Código CSS:

.contenedor {
  position: relative;
  width: 400px;
  height: 300px;
  background: #f0f0f0;
}

.elemento {
  position: absolute;
  bottom: 20px;
  right: 20px;
  background: #e74c3c;
  width: 150px;
  height: 60px;
}

Resultado Visual:

Contenedor (relative)
Elemento (absolute)

El elemento rojo está posicionado a 20px del borde inferior y derecho del contenedor

⚠️ Importante: Para usar position: absolute correctamente, casi siempre necesitas poner position: relative en el padre. Si no lo haces, el elemento absolute se posicionará relativamente a la ventana del navegador (muy probablemente no es lo que quieres).

3.4 Position: Sticky (Pegajoso)

Con position: sticky, el elemento se comporta como static hasta que alcanza una posición especificada en la ventana. Entonces se "pega" a esa posición.

📍 Características de Sticky
  • El elemento se comporta como static mientras se desplaza normalmente
  • Cuando alcanza la posición especificada (ej: top: 0), se "pega" a la ventana
  • Se mantiene pegado hasta que su padre sale de la vista
  • Muy útil para encabezados de tabla o menús de navegación
  • Requiere especificar top, bottom, left, o right

Código CSS:

.header {
  position: sticky;
  top: 0;
  background: #667eea;
  color: white;
  padding: 20px;
  z-index: 10;
}

Resultado Visual:

Position Flujo Normal Respecto a... Use Case (Caso de uso)

La propiedad Z-Index

Cuando elementos se superponen, z-index controla cuál está encima. Mayor número = más arriba.

Código CSS:

.caja1 {
  position: absolute;
  z-index: 10;
  background: #667eea;
}

.caja2 {
  position: absolute;
  z-index: 5;
  background: #e74c3c;
}

.caja3 {
  position: absolute;
  z-index: 1;
  background: #27ae60;
}

Resultado Visual:

z-index: 1
z-index: 5
z-index: 10

Mayor z-index = elemento más adelante. El azul (10) tapa al rojo (5) que tapa al verde (1)

4. Ejercicios Prácticos

A continuación encontrarás ejemplos para practicar selectores combinados y posicionamiento. Intenta resolver cada uno antes de ver la solución.

✏️ Ejercicio 1: Selectores Combinados - Menú

Objetivo: Cambia el color de los links en un menú usando selectores combinados.

HTML:

<nav class="menu">
  <ul>
    <li><a href="#">Inicio</a></li>
    <li><a href="#">Acerca</a></li>
  </ul>
</nav>

Requisitos:

  • Los links deben ser azules (#667eea)
  • En hover cambiar a blanco con fondo azul
  • Usa selector hijo (>)

Solución:

.menu ul > li > a {
  color: #667eea;
  text-decoration: none;
  padding: 10px 15px;
}

.menu ul > li > a:hover {
  background: #667eea;
  color: white;
}

Explicación: El selector .menu ul > li > a accede directamente a los links que son hijos de los items de lista. El > asegura que solo afectes los elementos directos.

✏️ Ejercicio 2: Position Absolute - Tarjeta con Badge

Objetivo: Crea una tarjeta con un badge "Oferta" en la esquina superior derecha.

HTML:

<div class="card">
  <div class="badge">Oferta</div>
  <h3>Producto</h3>
  <p>Descripción</p>
</div>

Requisitos:

  • .card: width 300px, position relative, background blanco
  • .badge: position absolute, top 10px, right 10px, background rojo (#e74c3c)
  • El badge debe tener texto blanco y redondeado

Solución:

.card {
  position: relative;
  width: 300px;
  background: white;
  padding: 20px;
  border: 1px solid #ddd;
  border-radius: 8px;
}

.badge {
  position: absolute;
  top: 10px;
  right: 10px;
  background: #e74c3c;
  color: white;
  padding: 5px 10px;
  border-radius: 20px;
  font-weight: bold;
}

Explicación: El padre (.card) debe ser position: relative para que el badge absolute se posicione dentro de él. Sin esto, se posicionaría relativamente a la ventana.

✏️ Ejercicio 3: Position Sticky - Header Flotante

Objetivo: Crea un header que se queda en la parte superior al desplazar.

HTML:

<header class="sticky-header">
  <h1>Mi Sitio Web</h1>
</header>
<!-- Contenido largo aquí -->

Requisitos:

  • position: sticky
  • top: 0
  • background azul (#667eea), texto blanco
  • z-index: 10

Solución:

.sticky-header {
  position: sticky;
  top: 0;
  background: #667eea;
  color: white;
  padding: 20px;
  z-index: 10;
  box-shadow: 0 2px 5px rgba(0,0,0,0.1);
}

Explicación: Con position: sticky y top: 0, el header se comporta normalmente hasta que alcanza la parte superior. Entonces se "pega" a la ventana. El z-index: 10 lo mantiene encima de otros elementos.
