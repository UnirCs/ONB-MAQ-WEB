Unidades y Valores CSS

Guía visual de las diferentes unidades de medida

1. Tamaños de Texto

Comparación de unidades (padre con font-size por defecto: 16px)

Este texto usa font-size: 16px — Tamaño fijo en píxeles

Este texto usa font-size: 1em — 1× el tamaño del padre = 16px

Este texto usa font-size: 1rem — 1× el tamaño del root = 16px

Este texto usa font-size: 100% — 100% del padre = 16px

Padre con font-size: 20px

Este es el texto del padre (20px)

Hijo con 1em = 20px (hereda del padre)

Hijo con 1rem = 16px (siempre del root)

Hijo con 1.5em = 30px (1.5 × 20)

2. Efecto Compuesto de em

Cuando usas em en elementos anidados, el tamaño se multiplica en cada nivel. Esto puede causar efectos no deseados.

Nivel 1: font-size: 1.2em

Calculado: 16px × 1.2 = 19.2px

Nivel 2: font-size: 1.2em

Calculado: 19.2px × 1.2 = 23px

Nivel 3: font-size: 1.2em

Calculado: 23px × 1.2 = 27.6px

Solución: Usar rem

Nivel 1: font-size: 1.2rem = 19.2px

Nivel 2: font-size: 1.2rem = 19.2px (mismo tamaño)

Nivel 3: font-size: 1.2rem = 19.2px (consistente)

3. Anchos (width)

Unidades de ancho

width: 300px — Siempre 300 píxeles
width: 50% — 50% del contenedor padre
width: 80vw — 80% del ancho de la ventana

↑ Redimensiona la ventana para ver cómo cambian % y vw

Mismo CSS en contenedor más estrecho (400px)

width: 300px — Sigue siendo 300px (¡se sale!)
width: 50% — Ahora es 200px (50% de 400)

4. Alturas y Unidades de Viewport

Comparación de alturas

height: 100px Fijo
height: 20vh 20% ventana
height: 50% 50% padre*

* Para que % funcione en height, el padre debe tener altura definida

5. Espaciado con em

Usar em en padding/margin hace que el espaciado sea proporcional al tamaño de texto del elemento.

Padding proporcional al font-size

font-size: 14px; padding: 1em;

Padding = 14px

font-size: 18px; padding: 1em;

Padding = 18px

font-size: 24px; padding: 1em;

Padding = 24px

6. Combinando Unidades con calc()

La función calc() permite mezclar unidades

width: calc(100% - 40px)

Ancho completo menos márgenes fijos

width: calc(50vw - 2rem)

Mitad de la ventana menos espaciado

font-size: calc(1rem + 0.5vw)

Texto fluido que crece con la ventana

7. Resumen: Cuándo Usar Cada Unidad

Unidad Relativa a Usar para
px Nada (absoluto) Bordes, sombras, detalles pequeños
rem Font-size del <html> Font-sizes, espaciados, ¡casi todo!
em Font-size del padre Padding/margin proporcional al texto
% Valor del padre Anchos flexibles, layouts
vw/vh Viewport (ventana) Secciones full-screen, texto fluido
